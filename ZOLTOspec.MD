What Zolto Must Support

1. Rich Text Notes

Like Markdown but more powerful.

Example:

# Motion and Force

[important]
Newton's First Law explains inertia.
[/important]

[tip]
Always draw free-body diagrams.
[/tip]

2. Mathematics System

This is mandatory.

Example:

@math
a² + b² = c²

or

$$
\int x^2 dx
$$

You should support:

Algebra

Geometry

Calculus

Graphs

Statistics

Matrices

Scientific formulas



---

3. Educational Drawings

This is where Zolto becomes unique.

Instead of importing images:

@diagram geometry
circle radius=5
triangle inside
label A top

or

@reasoning
square -> circle -> triangle
?
options: A B C D

This is VERY powerful.

4. MCQ System

Huge feature.

Example:

@mcq

A: Newton
B: Einstein
C: Tesla
D: Edison
answer: A

Advanced:

Timers

Explanations

Difficulty

Tags

Negative marking

Quiz mode

5. Reasoning Blocks

Your biggest innovation opportunity.

Example:

@reasoning sequence
2 4 8 16 ?

or

@analogy
Bird : Sky :: Fish : ?

or visual reasoning:

@visual-grid
□ ○ △
○ △ ?

No current markup language does this well.
